### This app is build with the following technologies:

1. NestJS (for backend)
2. MikroORM (for database)
3. PostgreSQL (for database in production)
4. SQLite (for database in development)
5. TypeScript (for type safety)
6. Jest (for testing)
7. Swagger (for API documentation)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
